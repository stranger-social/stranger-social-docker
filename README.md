## Quick Start

```bash
# Clone repository
git clone https://github.com/stranger-social/mastodon-docker
cd mastodon-docker

# Configure environment
cp .env.example .env
nano .env  # Edit with your settings

# Start services
docker compose up -d

# Check status
docker compose ps
```

## Common Commands

```bash
# Start all services
docker compose up -d

# View logs
docker compose logs -f web

# Create admin user
docker exec mastodon-web bin/tootctl accounts create username \
  --email=user@stranger.social --confirmed --role=Admin

# Database backup
docker exec mastodon-postgres pg_dump -U mastodon mastodon_production | gzip > backup.sql.gz

# Restart services
docker compose restart

# Stop services
docker compose down
```
